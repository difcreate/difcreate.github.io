102. Binary Tree Level Order Traversal-二叉树的层次遍历

描述：

给定一个二叉树，返回其按层次遍历的节点值。 （即逐层地，从左到右访问所有节点）。

例如:
给定二叉树: [3,9,20,null,null,15,7],

返回其层次遍历结果：

递归版：

结果数组输出格式：
    [
    [3],
    [9,20],
    [15,7]
